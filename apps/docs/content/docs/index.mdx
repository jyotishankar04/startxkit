StartXKit is a TypeScript CLI toolkit that scaffolds production-ready backend projects. Think `create-vite` but for backends.

## What It Does

- **Scaffolds a full project** with a single command: `npm create @startxkit@latest`
- **Generates modules** with routes, controllers, services, and repositories: `npx @startxkit/cli add module users`
- **Supports 3 frameworks**: Express, Fastify, and Hono
- **Supports 3 architectures**: Minimal, Modular, and Layered
- **Optional features**: CORS, env config, Zod validation, logging, error handler, Docker

## Quick Start

```sh
npm create @startxkit@latest
```

Follow the interactive prompts. In under a minute, you'll have a runnable backend.

## Project Structure

```text
my-api/
  src/
    app.ts              # App creation with middleware
    server.ts            # Entry point
    config/              # Env, CORS, etc.
    shared/              # Errors, middlewares, utilities
    modules/             # Feature modules
      health/            # Built-in health check
        health.route.ts
        health.controller.ts
    routes/
      index.ts           # Central route mounting
  startxkit.config.json  # Project config
  package.json
  tsconfig.json
```
